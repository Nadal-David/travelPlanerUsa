# Travel Planner USA

Projet statique de base en HTML, TailwindCSS et JavaScript.

## Demarrage

1. Installer les dependances:

```bash
npm install
```

2. Generer le CSS Tailwind:

```bash
npm run build
```

3. Ouvrir `index.html` dans un navigateur.

## Copie sur iPhone hors connexion

Apres `npm run build`, copie tout le dossier sur l'iPhone, par exemple dans l'app Fichiers, puis ouvre `index.html`.

Le site ne depend d'aucun serveur.
